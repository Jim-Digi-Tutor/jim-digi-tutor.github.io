Using a Mouse   →

These activities will teach you all about the Mouse, which is the main tool for using computers. You'll find out what each button does, and practice using the mouse correctly.

Using a Keyboard   →

In these activities you will learn all about the Keyboard, which is used for entering text into a computer, and also for some commands. You'll learn about some of the main keys and key combinations.

Files & Folders   →

Learn about Files & Folders and how they can be used to store our data and make the day-to-day use of a computer easier and more efficient.
