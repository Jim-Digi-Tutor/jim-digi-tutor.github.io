    <h1>Navigating Folders</h1>
          
    <h2>What is navigation?</h2>
          
    <p>
      When you look in folders, and move from one folder to another, it is known as <span class="emphasise">navigating</span>. If your job or study involves computers, you might sometimes be asked to navigate (or go) to a folder. This involves moving to the folder in question.
    </p>

    <h2>Folder paths</h2>

    <p>
      When you have to more than one folder to get where you need to be, it is known as a <span class="emphasise">folder path</span>. A folder path is a list of folder names that finish at a particular folder.
    </p>

    <p>
      So, if you have a folder called <span class="emphasise">Photos</span>, and that folder is in the folder <span class="emphasise">Documents</span>, the path to the Photos folder is <span class="emphasise">Documents &#8594; Photos</span>. 
    </p>

    <p>
      Don't worry if you don't understand folder paths straight away, you'll do some exercises that will help explain them.
    </p>
