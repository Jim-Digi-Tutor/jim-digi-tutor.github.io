Folder Structures

Home

A collection of folders arranged in a certain way is called a folder structure. The best way to set up your folder structure is in a way that works for you. A diagram of an example folder structure is shown below.

Folder structures start at the root, the actual physical device on which the data is saved. In this example, the root is a hard disk drive, but could also be a USB stick, a DVD, or cloud storage like OneDrive.

In this example, there are two folders at the first level Work Documents and Home Documents. These folders each contain two subfolders some of which themselves contain subfolders.
